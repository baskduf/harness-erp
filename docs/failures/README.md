# Failure Memory

Record a failure note here when a user-visible runtime failure, failed check,
repeated agent mistake, CI failure, or cross-environment mismatch needs durable
memory for future agents.

Do not invent fake failures. If a failure is recorded, include:

- What failed.
- How it was detected.
- The durable prevention check or why no practical check exists.
- The related task or commit.

Setup added this directory without fake failure records.
